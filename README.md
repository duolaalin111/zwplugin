# 图库管理组件 (LibraryManageAntdUI)

基于 **AntdUI** 的 WinForms 图库管理界面，支持产品图库结构树、图纸卡片分页展示、筛选、属性查看等功能。

## 功能特点

- 多产品切换（产品A / 产品B / 产品C）
- 树形结构浏览图库分类（如方案库 → 进线柜、基础图库、符号库等）
- 图纸卡片以流式布局展示，支持分页
- 按主属性筛选图纸（全部 / 进线柜类 / 母联柜类 / 补偿柜类）
- 点击图纸卡片展示详细属性（图名、所属库、尺寸、创建/修改时间）
- 右侧提供入库、出库、编辑、删除等操作按钮（演示用，可扩展实际逻辑）

## 技术栈

- .NET Framework 4.6.1+ 或 .NET 6/8
- WinForms
- [AntdUI](https://github.com/AntdUI/AntdUI) 2.3.2+ (第三方 UI 库)

## 使用方法

1. 引用 AntdUI 库（可通过 NuGet 安装：`Install-Package AntdUI`）
2. 将 `LibraryManageAntdUI.cs` 添加到您的 WinForms 项目中。
3. 从主窗体调用：
   ```csharp
   var frm = new LibraryManageAntdUI();
   frm.Show();
启动后选择产品，左侧树节点展开，点击叶子节点（如“进线柜”）查看图纸卡片。

数据扩展
当前使用硬编码的示例数据（InitializeProductData 方法）。您可以替换为实际数据源：

修改 productData 字典的构建逻辑，从数据库、JSON 文件或 Web API 加载。

LibraryNode 中的 Drawings 数组可改为存储图片路径、ID 等真实信息。

图纸卡片目前为自绘图形，您可将 Paint 事件中的图形替换为真实图片（Image.FromFile 或从资源加载）。
